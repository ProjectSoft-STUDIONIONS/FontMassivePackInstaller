===========================================================
FonTemp - история изменений
-----------------------------------------------------------

   + Добавлено
   * Исправлено
   - Удалено

вер. 1.7.1 (14.06.2014)
  * Версия x64: Исправлена ошибка при проверке повторного запуска FT (например добавление новых шрифтов из FontMassive при уже открытом окне FT).
  * Перевод интерфейса на English был не полным.

вер. 1.7 (09.03.2014)
  * Изменен домашний сайт пакета FontMassive Pack - http://fontmassive.com (см. также изменения в файлах Readme.txt и License.txt).
  * Исправлено сохранение/открытие файлов наборов (созданные пользователем новые не-Default наборы не открывались)
  * При переименовании наборов их шрифты «заменялись» шрифтами из текущего списка.
  + При добавлении новых шрифтов в текущий набор теперь обновляется счётчик шрифтов (около названия набора) и список шрифтов прокручивается к добавленным шрифтам.

вер. 1.6 (01.02.2014)
  + Теперь и x64!
  * Все файлы настроек сохраняются в UTF8.
  * Файлы настроек теперь сохраняются в папке «FT_Data» (файлы наборов остаются в папке «FTSets»).
  + Интеграция с FontMassive: для просмотра шрифтов можно использовать FontMassive 3.0.10 и выше (правая кн. мыши на списке наборов или списке шрифтов в FonTemp).

вер. 1.5.2.1 (23.03.2011)
  * Исправлено: при запуске с параметром «/min» не срабатывало подключение шрифтов.

вер. 1.5.2 (19.03.2011)
  + Добавлена возможность запуска в свернутом состоянии (параметр командной строки /min).
  * Исправлено открытие наборов .FTSet из Проводника (например, двойным щелчком по ним).

вер. 1.5.1 (08.03.2011)
  * Изменен механизм закрытия программы и отключение при этом шрифтов. Запрос на подтверждение отключения теперь выводится только при нажатии кнопки «Закрыть» либо из меню программы - все иные способы выхода не выдают запроса.

вер. 1.5.0.11 (15.01.2011)
  * Ошибки при удалении наборов.
  * Изменено окно создания нового набора.

вер. 1.5.0.0 (04.01.2010)
  FT переписан заново. Этим были устранены ошибки, связанные с русским языком и прочие.
  Слегка переделан функционал.
  Наборы шрифтов (файлы *.ftset) теперь хранятся во вложенной папке /FTSets - при первом запуске
  они будут перенесены туда автоматически из папки, где расположен FonTemp.exe.

вер. 1.4.9.1 (12.11.2007)
  * не работало перетаскивание шрифтов из Проводника на окно программы. (Зато работало из TotalCommander'а).

вер. 1.4.9 (05.11.2007)
  + Пункт меню «Выделить в Проводнике» - для выделенного шрифта в списке.
  * Исправление мелких ошибок.

вер. 1.4.8.1 (25.10.2007)
  * Не открывались некоторые шрифты Type1

вер. 1.4.8 (22.10.2007)
  * Переделан метод чтения ресурсов из шрифтовых файлов, что избавило от большого количества ошибок в работе программы.
  + Окно «Редактора наборов» - изменяемого размера.

вер. 1.4.7 (12.09.2007)
  + Запрос на подтверждение при попытках закрытия шрифтов из списка.
  - Убрана кнопка «Закрыть» для главного окна во избежание случайного закрытия. Используйте Alt+F4 или меню.
  * Перекрытие окна сохранения при создании нового набора.
  * Переработано меню.

вер. 1.4.6.1 (27.08.2007)
  * Невернная обрисовка заголовка окна.

вер. 1.4.6 (23.08.2007)
  * Иногда не добавлялись шрифты, добавленные в командной строке (т.е. НЕ перетескиванием на окно).
    (Точнее, для «видимости» нужно было их «переВключить»).
  * При первом вызове после установки неверно создавался набор по-умолчанию.
  * Легкий ре-дизайн редактора наборов.
  * Заменено окно «О программе».
  * Двойной заголовок (иногда) при выключенной теме WinXP.

вер. 1.4.2 ..... 1.4.5 (07.08.2007)
  Изменений нет - номер версии увеличен с учетом версии всего пакета.

вер. 1.4.1 (27.06.2007)
  * Иногда «баловался» гориз. бегунок.

вер. 1.4 (24.06.2007)
  * После добавления шрифтов   список иногда не реагировал на «тычки» мышью.
  + Запрет на повторное добавление шрифтов в список. В случае, если добавляемый шрифт уже есть в списке - у него включается маркер («квадратик»).
  * Цвет маркера изменен на синий (чтобы не путать с FontMassiv'овским).
  * Убран «Притормоз» при открытии/закрытии списка (запуске/закрытии программы).
  * При изменении списка адекватно реагирует редактор наборов.
  + В заголовке главного окна указан текущий набор.
  + Пункт меню «Наборы шрифтов» в контекстном меню списка.
  * Перерисовано главное окно.
  + При запуске с параметром «-selset»  (в FontMassive пункт меню «FonTemp (выбрать набор)» ) в редакторе наборов появляются кнопки «Отмена» и «OK»,
    а также меняется заголовок окна: «Выберите набор или создайте новый».
  * Ошибка: дубликация файлов в окне наборов.
  * Не применялось измененное название набора.
  + Файлы из списка шрифтов можно перетаскивать на другие программы.

вер. 1.3.0.0 BETA (21.05.2007)
  + Добавлены «Наборы шрифтов» - т.е. группы для использования в разных проектах.
  * Улучшено поведение маркеров и работа при добавлении/удалении шрифтов в список (перенос мышью).
  [!] Beta: Возможно, теперь пока неверно работает запуск с командной строкой.
