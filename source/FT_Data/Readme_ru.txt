FonTemp служит для временной установки шрифтов независимо от того, где они находятся.
Шрифты становятся видны для всех программ, но только пока запущена FonTemp, либо
до перезагрузки системы.

Это удобно, когда Вы работаете над каким-либо проектом, для которого нужны
редко используемые шрифты, а устанавливать их в систему «на постоянно» нет необходимости.

FonTemp является частью пакета FontaMassivePack (совместно с менеджером шрифтов FontMassive, определителем шрифтов по изображению FontDetect).
Эти программы находятся здесь: http://fontmassive.com/ru/

Примечания:
1. Для формирования списка перетащите файлы шрифтов на окно программы.

2. Некоторые программы не обновляют свой список шрифтов после их добавления (независимо от метода)
    В этом случае такую программу необходимо просто перезапустить.

3. Запуск в свернутом виде (например автозапуск при старте системы) - используйте параметр "/min"
    Пример: FonTemp.exe /min
    ("/min" можно добавить в  свойствах ярлыка)
