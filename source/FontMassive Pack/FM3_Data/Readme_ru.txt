=====  Возможные параметры командной строки  =====

1) Открыть папку, файл или список из файла
FontMassive.exe Полный_путь_к_папке
FontMassive.exe Полный_путь_к_файлу_шрифта
FontMassive.exe Полный_путь_к_списку_fml
FontMassive.exe Полный_путь_к_списку_ftset

Можно указать несколько файлов или папок.
Если есть пробелы - следует заключить в двойные кавычки.


2) Типы файлов
Указаине расширений файлов, с которыми будет работать FM
FontMassive.exe /fext=*.ttf;*otf;*.pfm

Например, по умолчанию отключена работа с растровыми .fon
Включить их (и только их):
FontMassive.exe /fext=*.fon

Добавить с другими типами:
FontMassive.exe /fext=*.ttf;*otf;*.pfm;*.fon


3) Выделить
Выделить шрифты при запуске

Выделить по номерам в списке (нумерация с 1)
FontMassive.exe Полный_путь_к_папке /sel_n=3,7,34
Разделитель - запятая

Выделить по части имени файла
Пример - выделить все *arial* или *script*
FontMassive.exe Полный_путь_к_папке /sel_f=arial,script
Разделитель - запятая


---- Параметры, используемые самим FM в работе -----

4) Установка шрифтов из файла списка
FontMassive.exe /installfonts="путь_к_файлу_списка"
Формат текста в файле - UTF-8
Запускать - от имени Администратора
Результат работы - ErrorLevel
1000   = ошибка
Больше 1000 - количество установленных из списка (1002 - два шрифта)

Пример cmd-файла:
FontMassive.exe /installfonts=list.txt
@echo %ERRORLEVEL%

5) Удаление шрифтов из файла списка
Удаляются все сопутствующие указанным шрифтам файлы (для *.pfm будут удалены также *.pfb и т.д.).
FontMassive.exe /delfiles="путь_к_файлу_списка"
Формат текста в файле - UTF-8
Запускать - от имени Администратора
Результат работы - ErrorLevel
1000   = ошибка
Больше 1000 - количество удалённых из списка (1002 - два шрифта)
Также в исходном файле списка остаются те записи, при удалении которых возникла ошибка
(успешно удалённые из списка удаляются)
